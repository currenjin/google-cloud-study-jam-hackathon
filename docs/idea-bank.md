# Idea Bank

## 1. Small Table Matcher — 추천

**Problem:** 작은 모임에서 깊은 대화가 자연스럽게 열리기 어렵다.

**User:** 모임 진행자, 커뮤니티 리더, 스터디 운영자.

**MVP:** 참가자 정보와 모임 목적을 넣으면 Gemini가 대화 흐름과 질문을 설계한다.

**Google Cloud Fit:** Gemini API, Cloud Run, 향후 Firestore/Sheets 연동.

## 2. Reflection Coach

하루 기록을 입력하면 감정, 가치, 다음 작은 실천으로 정리한다.

## 3. Study Jam Buddy

학습 내용을 입력하면 복습 퀴즈, 요약, 다음 실습 체크리스트를 만든다.

## 4. Document-to-Action

문서/PDF 내용을 요약하고 액션아이템으로 변환한다. 기술 시연성은 좋지만 시간 리스크가 있다.
