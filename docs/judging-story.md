# 3-Minute Judging Story

## One-liner

Small Table Matcher는 Gemini로 작은 모임의 대화 흐름을 설계해, 사람들이 더 안전하고 깊게 연결되도록 돕는 서비스입니다.

## 0:00-0:30 Problem

좋은 모임은 사람이 모였다고 자동으로 생기지 않습니다. 진행자는 누가 어떤 상태인지, 어떤 질문부터 던져야 하는지 매번 고민합니다.

## 0:30-1:00 Solution

참가자 정보와 모임 목적을 입력하면 Gemini가 아이스브레이크, 깊은 질문, 소그룹 구성, 마무리 액션을 제안합니다.

## 1:00-2:20 Demo

1. 참가자 3명의 관심사/고민 입력
2. 모임 목적 입력
3. 생성 버튼 클릭
4. 진행자용 대화 흐름 확인

## 2:20-2:50 Google Cloud Fit

Gemini API로 맥락 기반 대화 설계를 생성하고, Cloud Run으로 빠르게 배포했습니다. 향후 Google Sheets/Calendar와 연동하면 모임 준비와 회고까지 자동화할 수 있습니다.

## 2:50-3:00 Closing

AI를 더 많은 콘텐츠 생산이 아니라, 사람들이 서로를 더 잘 듣고 돕는 방향으로 쓰고 싶었습니다.
