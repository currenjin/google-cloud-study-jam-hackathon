# 해커톤 시작 전에 해놓을 것

규정상 코드는 현장에서 작성해야 하므로, 사전 준비는 **환경 세팅 + 설계 암기 + 로지스틱스**까지만.

## 1. 환경 세팅 (집/사무실에서, 현장 와이파이 믿지 말 것)

- [ ] **Antigravity 설치** — <https://antigravity.google/download>
  - 현장에서 @gcplab.me 계정으로 로그인. **로그인 시 프로젝트 ID를 지정해야 Business Plan 적용됨** — 이거 빼먹으면 무료 플랜으로 돌아감
- [ ] **Chrome 새 프로필(또는 시크릿 창) 준비** — 임시 계정(@gcplab.me)은 개인 구글 계정과 분리해서 로그인
- [ ] **GitHub 인증 확인** — `gh auth status`. 현장에서 새 public repo 생성 → 즉시 푸시 가능해야 함
- [ ] **화면 녹화 리허설** — QuickTime 화면 녹화(⌘⇧5) 한 번 돌려보고 마이크 입력 확인. 데모 영상 제출 필수
- [ ] **노트북 + 충전기** 챙기기 (멀티탭은 현장 제공)
- [ ] Node.js / 에디터 등 기본 개발 환경 동작 확인

## 2. 설계 암기 (문서 정독)

- [ ] [03-prompts.md](03-prompts.md)의 Story Bible 스키마와 프롬프트 3종을 읽고 구조를 머리에 넣어둔다 — 현장에서 프롬프트를 고민하는 시간이 제일 아깝다
- [ ] 화면 3장 구성을 손으로 한 번 그려본다:
  1. **시드 입력** — 텍스트로 드라마 시작 (이미지 시드는 해커톤 이후)
  2. **턴 화면** — 현재 컷 + "다음 전개 한 줄" 입력 + "다음 사람에게 넘기기" 버튼
  3. **완성 릴 재생** — 9:16 세로, 자동 넘김
- [ ] 데모 시나리오(시드 1개 + 릴레이 전개 3개) 확정 — 03-prompts.md 참고

## 3. 로지스틱스

- [ ] **점심 먹고 입장** — 현장은 다과·커피만 제공, 식사 없음
- [ ] **대중교통** — 주차 지원 불가. 3호선 동대입구역 5번 출구 도보 5분 (호텔 게이트→로비 무료 셔틀 있음)
- [ ] 본관 1층 로비 → **연회장 전용 엘리베이터** → 3층 → '오키드룸' 표지판 따라 이동
- [ ] 12:10–12:30 체크인. 늦지 않게 12:00 도착 목표
- [ ] (선택) 오전 10–12시 영빈관 **Gemini Playground 투어** — 반드시 오키드룸 앞 등록 데스크에서 팔찌 먼저 수령

## 4. 현장 도착 직후 (개발 시작 전)

> 툴이 처음이면 [05-tools-guide.md](05-tools-guide.md)를 먼저 읽는다 — 어떤 프로젝트를 선택하는지, 화면 어디를 누르는지, 흔한 에러 대응까지 정리되어 있다.

- [ ] @gcplab.me 계정 ID/PW 수령 → 시크릿 창에서 <https://console.cloud.google.com> 로그인 → 할당된 프로젝트(1개뿐) 선택 + **프로젝트 ID 메모**
- [ ] <https://ai.studio> 로그인 → "Get API key" → 키가 안 보이면 "Import Projects"로 프로젝트 선택 → API 키 확보
- [ ] AI Studio 모델 목록에서 텍스트/이미지 모델 ID 확인 (AGENTS.md의 모델명과 다르면 에이전트에게 교체 지시)
- [ ] Antigravity 로그인 (프로젝트 ID 지정!) → **이 레포 클론해서 폴더 열기** (AGENTS.md 자동 로드)
- [ ] `cp .env.example .env` → API 키 입력
- [ ] 행사 키는 localhost에서만 사용하고 공개 배포하지 않는다. `.env` 커밋 금지, 행사 종료 직후 키 폐기
- [ ] API가 불안정하면 `.env`의 `VITE_DEMO_MODE=true`로 전체 데모가 동작하는지 확인
- [ ] 13:20 시작과 동시에 [04-kickoff-prompts.md](04-kickoff-prompts.md)의 M1 프롬프트 붙여넣기
