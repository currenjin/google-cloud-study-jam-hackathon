# Stage 00 — GCP Warmup

목표: 해커톤 당일 GCP 콘솔/Cloud Shell에서 헤매지 않기.

## 체크리스트

- [ ] Google Cloud Console 로그인
- [ ] 프로젝트 선택/생성
- [ ] Billing/크레딧 상태 확인
- [ ] Cloud Shell 열기
- [ ] `gcloud auth list` 실행
- [ ] `gcloud config get-value project` 실행
- [ ] 필요한 API 후보 확인: Cloud Run, Artifact Registry, Generative Language API 또는 Vertex AI

## 메모

- UI/문서는 가능하면 영어로 사용한다.
- API enable, IAM, region 설정에서 시간이 많이 날아간다.
