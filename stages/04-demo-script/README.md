# Stage 04 — Demo Script

목표: 3분 발표를 외울 정도로 단순화한다.

## 구조

1. 문제: Study Jam 참가자는 배운 뒤에도 약점과 다음 행동이 흐려진다.
2. 해결: Gemini가 학습 메모를 복습 퀴즈/약점 분석/실습 액션으로 바꾼다.
3. 데모: 학습 메모 입력 → 학습 코치 생성 → 퀴즈와 체크리스트 확인
4. 확장: Sheets/Classroom/Calendar와 연결해 스터디 복습 루틴 자동화

## 완료 기준

- 3분 안에 발표 가능
- 앱이 죽어도 fallback 출력/스크린샷으로 설명 가능
